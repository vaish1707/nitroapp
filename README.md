## Installation

* npm install http-server
* cd nitroapp
* $nitroapp > http-server

```javascript
C:\nitroapp>http-server
Starting up http-server, serving ./
Available on:
  http://127.0.0.1:8080
Hit CTRL-C to stop the server
```
* Open index.html
